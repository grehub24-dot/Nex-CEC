## Usage

1. Pick a brand from the list above
2. Copy its `DESIGN.md` to the project root:
   ```
   copy docs\awesome-design-md\design-md\supabase\DESIGN.md DESIGN.md
   ```
3. Tell the agent: *"Use DESIGN.md and retheme this page"*
4. Agent reads the tokens and generates matching CSS

## Files

Each brand folder contains:
- `DESIGN.md` — Design system definition (colors, type, components, principles)
- `README.md` — Quick preview
- `preview.html` — Visual catalog (in some folders)
